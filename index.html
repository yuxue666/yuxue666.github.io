How to build your own blog

Posted on 2021-05-07 Edited on 2021-05-08

1. Preparation

1.1 Install hexo

System requirement: MacOS

Input your password for Mac, then start to install Hexo. You can also find installation guide on Hexo. First, install Node.js. Check node and npm version.

Use npm to install npmc.

Then use cnpm to install hexo.

-g means globally.

2. Create your first blog

2.1 Hexo project

First, Create a folder in your computer to store your hexo project. In that floder, initiate a hexo proect using command as follows.

Start the hexo project you just create.

The output remind you go to local host (http://localhost:4000) to check your hexo website.

2.2 Hexo blog

Write your blog in folder ./source/_posts/

You can use Typora or even vim or any text editor to write your blog in Markdown format (file name ends with .md). Check Markdown grammer here or here.

Generate and start your hexo blog.

3. Put your hexo blog pojext to github

Sign in your github account and create a new repositary. My repositary name is “yuxue666.github.io”. Go to the hexo peoject folder on your computer and install hexo-deployer-git.

Configure in file ./_config.yml

Go to the end of _config.yml. Change Deployment block into:

# Deployment

## Docs: https://hexo.io/docs/one-command-deployment

deploy:

type: git

repo: https://github.com/yuxue666/yuxue666.github.io.git

branch: master

Then deploy. Need to input your github Uername and Password.

4. Change theme of your hexo blog.

Yilia Theme

Configure in file ./_config.yml

Go to the end of _config.yml. Change Extensions block into:

# Extensions

## Plugins: https://hexo.io/plugins/

## Themes: https://hexo.io/themes/

theme: yilia

Generate and start your hexo blog. Then deploy.
